1)What git command creates a branch?
git checkout -b

2) What is the difference between a fast-forward and recursive merge?
Fast forwards happen when there have not been commits on the original branch while the new branch is being worked on.


3) What git command changes to another branch?
git checkout


4) What git command deletes a branch?
git branch -D


5) How do merge conflicts happen?
When Git can not determine what file or folder to choose when merging.
